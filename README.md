# Library Management System using Java

## Project Overview
This is a simple console-based Library Management System developed using Java. The project helps users manage books through a menu-driven interface.

## Features
- Add Books
- View Books
- Search Books
- Exit System

## Technologies Used
- Java
- OOP Concepts
- Arrays
- Scanner Class
- Conditional Statements
- Loops

## How to Run

### Compile
```bash
javac LibraryManagement.java
```

### Run
```bash
java LibraryManagement
```
